# Acerca de la Libreria SiatClient

La libreria le permite conectar y consumir los servicios de la API Rest de Facturacion de SinticBolivia (MonoInvoices).

Para poder consumir la API, debe tener una cuenta y credenciales de acceso.

Puede solicitar las credenciales en nuestro sitio web.

http://sinticbolivia.net

# Execute Test
In order to execute the tests, you need to go to parent directory of this module, and execute the next command

```
python -m client.test
```
